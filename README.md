
## install project

### command
- git clone repoLink 
- composer install
- create database megex and set .env file
- php artisan migrate 
- php artisan db:seed 
- php artisan serve 
## Requirements Laravel

- PHP version 8.1
- Mysql DB 

## End Points 
 -- /admin  to config home slider 
 -- {/products}   to crud  product resource [/create,products/:id,products/:id/edit]
